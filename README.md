# Osteoporosis Risk Prediction
A project for predicting osteoporosis risk using healthcare and lifestyle data.


## Overview

This project focuses on predicting osteoporosis risk using healthcare, demographic, lifestyle, and medical history data. The goal is to analyze risk factors, extract useful insights, and build machine learning models to support early identification of individuals at higher risk of osteoporosis.


## Features

- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical analysis and hypothesis testing
- Feature encoding
- Logistic Regression model
- Decision Tree model
- Model evaluation and comparison


## Technologies

- Python
- Pandas
- NumPy
- SciPy
- Scikit-learn
- Matplotlib


## Dataset

The dataset contains healthcare and lifestyle information related to osteoporosis risk factors, including age, gender, hormonal changes, family history, body weight, calcium intake, vitamin D intake, physical activity, smoking, alcohol consumption, medications, prior fractures, and medical conditions.


## Results

The analysis showed that age and hormonal changes were among the most important factors associated with osteoporosis risk.
